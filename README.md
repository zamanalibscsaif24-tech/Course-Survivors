# 🎓 Course Survivors

**Course Survivors** is a community-driven course review and academic insight platform built for students of **Sukkur IBA University**. The platform enables students to share course experiences, evaluate workload and difficulty, and make informed academic decisions through peer-generated feedback and analytics.

## Overview

Selecting courses can be challenging when students have limited information about course difficulty, workload, teaching quality, and overall learning experience. Course Survivors addresses this problem by creating a centralized platform where students can contribute reviews, explore course insights, and learn from the experiences of their peers.

The application combines community engagement, academic analytics, and a modern user experience to support better course planning throughout the BS Computer Science curriculum.

---

## Key Features

### 📊 Course Analytics

* Aggregated ratings for overall course quality and difficulty
* Multi-dimensional review system covering:

  * Course Difficulty
  * Workload
  * Instructor Effectiveness
  * Overall Experience
* Visual indicators and progress bars for quick interpretation
* Course statistics based on real student feedback

### 🎮 Gamified Student Engagement

* Reward-based contribution system
* Points earned for submitting course reviews
* Dynamic leaderboard showcasing top contributors
* Personal activity and contribution tracking

### 🔍 Smart Search & Filtering

* Search courses by code or title
* Filter by semester, category, and department
* Persistent filter states for improved navigation
* Shareable filtered views through clean URL parameters

### 📚 Complete BSCS Curriculum Coverage

The platform includes the full Sukkur IBA BS Computer Science curriculum, covering:

* Computing Core Courses
* Artificial Intelligence Specialization
* Mathematics and Statistics
* Cybersecurity and Emerging Technologies
* Domain Electives and General Education Courses
